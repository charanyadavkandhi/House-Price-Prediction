# 🏠 House Price Prediction Using Machine Learning

## 📌 Overview

House price prediction is a critical application of Machine Learning in the real estate industry. This project develops a predictive model that estimates residential property prices based on housing features and market-related attributes.

The project includes data preprocessing, exploratory data analysis, feature engineering, model training, and performance evaluation using regression algorithms.

---

## 🎯 Objectives

* Predict house prices using Machine Learning
* Analyze factors influencing property values
* Perform data cleaning and preprocessing
* Build and evaluate regression models
* Visualize housing market trends
* Improve prediction accuracy through feature engineering

---

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/charanyadavkandhi/House-Price-Prediction.git
```

Navigate to the project directory:

```bash
cd House-Price-Prediction
```

Install dependencies:

```bash
pip install pandas numpy matplotlib scikit-learn
```

Run the project:

```bash
python house_price_prediction.py
```
