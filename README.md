# House Price Prediction using Machine Learning

## Overview

This project predicts house prices using Machine Learning techniques. A Linear Regression model is trained on the California Housing Dataset to estimate house prices based on housing features.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Linear Regression

## Dataset

California Housing Dataset from Scikit-learn.

## Workflow

1. Load Dataset
2. Data Exploration
3. Train-Test Split
4. Linear Regression Model Training
5. Prediction
6. Model Evaluation using MSE and R² Score
7. Visualization of Actual vs Predicted Prices

## Results

* Mean Squared Error (MSE): 0.556
* R² Score: 0.576

## Output

The model predicts house prices and visualizes Actual vs Predicted House Prices.
